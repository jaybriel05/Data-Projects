## Introduction
This dashboard was created to help job seekers have an overview of the salaries for their desired data profession.

This data contains information on job titles, salaries, locations, and skills. There are many types of datasets in kaggle.com where you can find similar datasets.
### Dashboard file
The final dashboard file is available here: [Data Salary Calculator.xlsx](Data%20Salary%20Calculator.xlsx)

### Excel Skils used:
These are the skills I used to make the dashboard
- Data Validation
- Charts
- Formulas and Functions

### Dataset
The dataset contains realworld data professions informatioin starting from 2020 and includes detailed information on:
- Job Titles
- Salaries
- Locations
- Skills
